How this works

About

Pick a local folder, see all .mp3 files in a Finder-like list, click to view/edit ID3 metadata in the right pane (including embedded album art), and save changes back to disk. Multi-select to apply edits to many files at once.

How to use

Click Choose folder…, grant read/write access. Click any file in the left pane to inspect — fields appear on the right. Edit, then click Save. Hold ⌘/Ctrl or Shift to select multiple files; only fields you fill will be applied to all selected.

Technical details

Uses the showDirectoryPicker() File System Access API. ID3v2.3 parser/writer is implemented from scratch — reads/writes the leading tag, preserves audio data verbatim. Album art uses APIC frames. Text frames are written as UTF-16 LE with BOM for unicode safety. Multi-select uses placeholders (—) for fields with mixed values across the selection; only fields you actually edit are applied. Requires Chrome/Edge (Safari/Firefox lack the File System Access API).
