How this works

About

Loads a .3mf file entirely in the browser (nothing is uploaded), splits it into its constituent parts, and lets you preview the model in every combination of up to four Bambu Lab PLA Basic filament colors.

How to use

Drop a .3mf file onto the drop zone (or click to browse). Click swatches in the palette to pick up to four filament colors. The grid below shows every assignment of those colors to the model's parts; click any thumbnail to load it into the big interactive viewer. Drag to orbit, scroll/pinch to zoom.

Technical details

The 3MF is a ZIP archive — the Central Directory is located via the PK\x05\x06 end record (with ZIP64 fallback) and entries are inflated with DecompressionStream('deflate-raw'). Every .model file in the archive is parsed; each <object> element with geometry becomes a separately colorable "part" (falling back to <mesh>-level splitting when no objects are present).

Geometry is centered and scaled once over the combined bounding box so parts stay positioned correctly relative to each other. Vertex normals are computed by accumulating cross-product face normals per vertex. Two WebGL contexts are used: one driving the interactive canvas, and one 240×240 offscreen canvas (created with preserveDrawingBuffer: true) that re-renders each combination and exports the result via toDataURL() into a thumbnail image.

For P parts and K selected colors, there are KP assignments. The grid caps display at 32 thumbnails (you'll see a message if more exist), since the count grows quickly — pick fewer colors or upload a simpler model to see them all.
