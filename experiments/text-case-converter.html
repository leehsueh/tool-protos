How this works

About

Converts any text into common casing formats used in programming and writing. Useful when you need to quickly switch between naming conventions or format output text.

How to use

Type or paste text into the input box, then click any format button. The converted text appears below and can be copied with the copy button.

Technical details

Each conversion is a pure JS function operating on the raw input string. The core helper words(str) splits input into an array of lowercase words by splitting on whitespace, underscores, hyphens, and camelCase/PascalCase boundaries (via a regex). Every format function then joins words differently.

No libraries are used — just String.prototype methods and regular expressions. The copy button uses the navigator.clipboard API with a brief visual feedback state.
