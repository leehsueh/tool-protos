How to add an experiment

  1. Copy experiments/_template.html to experiments/your-experiment-name.html
  2. Fill in the <title> and <meta name="description"> tags
  3. Write your experiment in the <main> block and document it in the <details> section
  4. Add an entry to the EXPERIMENTS array in index.html — this is what makes it appear in the index
  5. Commit and push to main — GitHub Actions deploys it automatically
